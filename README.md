# MekanBul Backend Projesi

Bu proje, Web Teknolojileri ve Programlama dersi kapsamında geliştirilen MekanBul uygulamasının sunucu tarafıdır. Node.js, Express.js ve MongoDB teknolojileri kullanılarak RESTful API mimarisinde geliştirilmiştir.

## Kullanılan Teknolojiler

- Node.js
- Express.js
- MongoDB
- Mongoose
- MongoDB Atlas

## Kurulum ve Çalıştırma

### Gerekli Paketleri Yükleyin
npm install

###Sunucuyu Başlatın
npm start

Başarılı çalıştırma sonrası terminalde şu çıktıyı göreceksiniz:
Mongoose mongodb+srv://eneskocaslan:bubirdenemesifresi@cluster0.iswsksq.mongodb.net/mekanbul?retryWrites=true&w=majority adresindeki veritabanına bağlandı.
